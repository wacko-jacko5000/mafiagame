# Vision

## Product statement

Build a live-service mafia strategy platform that captures the pacing, tension, and social dynamics of classic browser mafia games while using a modern codebase that can support web, admin, and future mobile clients.

## Experience pillars

- Persistent criminal identity progression.
- Short-session actions with long-tail strategic planning.
- Social rivalry through gangs, territory, and market pressure.
- Clear feedback loops for risk, cooldowns, profit, and punishment.
- Live operations support through admin tools, realtime events, and configurable balance.

## Platform principles

- Backend API owns game state and rules.
- Shared packages define contracts, names, and balance catalogs.
- UI presents state and intent, not hidden formulas.
- Every domain module is independently documented and testable.
