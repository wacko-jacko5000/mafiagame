# Gangs Test Plan

- Unit: gang-name validation and leader/member rule handling.
- Application: create, join, leave, invite, accept, decline, and list flows.
- API: create gang, join gang, leave gang, invite player, accept invite, decline invite, get gang summary, list members, and list invites.
- Edge: duplicate gang names, duplicate joins, one-gang-per-player enforcement, one-pending-invite-per-player enforcement, reused invites, and leader leave with remaining members.
